Quick Start

To use KeyTik, you just need to install AutoHotkey and Interception driver. Interception driver is optional if you want to use KeyTik assign profile on specific device feature. Follow this guideline to do it:

  1. Install KeyTik.
  2. Find out how to use KeyTik automation tool.
  3. Find out what key supported on KeyTik.
