Create Automation Tool

On this page

  1. Using AI
  2. Download it Online
  3. Make Yourself

There are various ways to make an profile to do automation tool you want. Here is how you can do that.

Using AI #

You can use AI to made you an AutoHotkey script to do automation task you need. The key is, you need to made the automation task on AuotHotkey language v2. Then you can just copy the code provided with text mode and test it.

We also think to use python langeuge to run the profile because AI tent to use Python language if you ask it to made code. We still think about it and it may or may not implemented. Stay toon for the update.

Download it Online #

There are multiple pre-made AutoHotkey script online. Some platform like GitHub or AutoHotkey community has various code that can be used. If you find one, you can download it then use import function to add it into KeyTik.

Here is some download place we recommended:

  • (Outdated) GitHub Repository - Awesome AutoHotkey: List of selected AutoHotkey script for you to use.
  • AutoHotkey Community - Scripts and Functions (v2): Section to share working AutoHotkey script by community.

Make Yourself #

You can also learn AutoHotkey language and made the script yourself. This way, you can customize it more to your preference. If you feel stuck, you can always ask community about your code. There are a lot of AutoHotkey community you can find online.

Here is some of popular one:

  • AutoHotkey Community Official Website: Official community of AutoHotkey.
  • AutoHotkey Subreddit: AutoHotkey community on Reddit.
